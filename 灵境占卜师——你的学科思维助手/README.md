# 灵境占卜师——你的学科思维助手

## 📌 项目概览

**参赛主题**: 学科与校园文化
**作品性质**: 网页设计大赛参赛项目
**核心定位**: 用学科思维给校园困惑提供可执行建议的交互式占卜助手

---

## 🎯 设计理念

### 问题与解决方案

**问题背景**:
- 学生在学业、生活、情感方面面临困惑，但往往缺乏系统化的思维框架
- 各学科知识相对独立，难以跨域应用于实际生活问题
- 陌生的学科概念缺乏情感联系，学以致用的转化率低

**核心解决方案**:
通过"占卜师"这一拟人化角色，将7个学科的核心思维方法论转化为可落地的人生建议，降低理论学习门槛，让知识呈现更有温度、更有行动价值。

### 设计特色

1. **占卜叙事** - 用"占卜"这一文化符号做切入，降低学术严肃感，增加交互趣味性
2. **多角度思维** - 同一个问题支持从7个不同学科视角解读，培养跨学科思维习惯
3. **从理论到行动** - 每条建议都配有具体可执行的方法（如25分钟专注块、间隔重复、设置默认路径等）
4. **校园文化融合** - 所有场景都基于校园真实困境：拖延、社交恐惧、考试焦虑、职业迷茫

### 2. 交互设计

**打字机效果**
- 字符逐个输出，增强代理感与可读性
- 使用50ms间隔控制速度，视觉流畅

```javascript

  contentEl.textContent += data.text.charAt(index);

```

**智能学科切换追踪**
- 记录每个问题已用过的学科ID
- 避免同一问题重复获得相同学科建议
- 当所有学科都用完时提示"所有视角都看过了，再来一轮？"

```javascript
function getRemainingCount(usedSubjects) {
  const unique = new Set(usedSubjects);
  return Math.max(0, subjects.length - unique.size);
}
```

**聊天历史持久化**
- 用localStorage保存完整聊天记录
- "🗑️ 清空"按钮可重置所有状态
- 支持页面刷新后恢复对话

### 3. 内容系统设计

**7学科 × 15条建议 矩阵**
- 共105条独立建议
- 每条建议都包含：理论概念 + 生活映射 + 可执行行动

**示例（数学视角·拖延问题）**:
```
理论: "拖延像目标函数未收敛"
映射: 把任务拆成最小步骤，每步只追求局部最优
行动: "每次只做一个最小单元，晚间复盘误差并迭代"
```

**内容质量特征**:
- 避免空泛鸡汤，每条建议都有学科背书
- 强调可测量、可执行
- 针对校园真实场景：考试、社交、职业规划

## 📱 功能模块详解

### 模块1: 占卜聊天室 (index.html)

**核心组件**:
```
┌─────────────────────────┐
│    头部 (site-header)    │
│  🐱 灵境占卜师           │
│  学科占卜师 + 对话Agent  │
├─────────────────────────┤
│  聊天历史区 (chat-history) │
│  [用户消息]              │ ← 蓝色气泡，右对齐

│  输入区 (chat-input-row) │
```

4. 使用打字机效果逐字显示
5. 显示学科签名和"换一个学科"按钮
6. 所有内容自动通过localStorage保存

## 💾 状态管理与性能

### localStorage 数据结构
```javascript
STORAGE_KEY = "lingjing_chat_state_v2"

// 每条消息数据结构
{
  role: "user" | "agent",
  text: "消息内容",
  subjectId: "math" | "physics" | ...,  // Agent专有
  question: "用户原始问题",              // Agent专有
  questionId: "q_timestamp_random",      // 追踪同一问题的不同学科回复
  usedSubjects: ["math", "physics"],     // 已用过的学科ID集合
  allowSwitch: true                      // 是否显示切换按钮
}
```

### 性能优化
- **Canvas粒子**：根据设备视口自动调整数量 (最多80个)
- **事件节流**：鼠标移动使用requestAnimationFrame去抖
- **定时器管理**：activeTimers Set追踪并清理打字机定时器，避免内存泄漏
- **CSS优化**：使用will-change提示浏览器优化背景动画

## 🎓 学以致用体现

### 什么是"学科思维"？

不是记住公式或定义，而是学会用各学科的**顶层思想模型**来看待现实问题：

1. **数学** → "很多复杂问题可以转化为可追踪的变量与约束"
2. **物理** → "系统有规律，小的持续作用能产生大改变"
3. **计算机** → "把大问题分解成小问题，然后系统地解决"
4. **设计** → "用户体验和视觉秩序会影响行为"
5. **语言学** → "我们用的词汇会影响我们的思维"
6. **心理学** → "认识自己的认知规律，就能调节行为"
7. **生物学** → "生命是自我调节的系统，要尊重它的规律"

### 设计验证

用户在使用本工具时，会：
- 看到"拖延 = 目标函数未收敛" (数学抽象化)
- 看到"焦虑 = 能量滞留" (物理能量转化)
- 看到"困惑 = 参照系不清" (多学科多视角)
- 最后得到一句"我可以做..." (可执行行动)

关于本项目

网页设计大赛：学科与校园文化

比赛说明

“灵境占卜师——你的学科思维助手”是一个纯前端静态项目，围绕“学科与校园文化”主题，把抽象学科知识转化为轻量对话体验。用户可以在数学、物理、计算机、设计、语言学、心理学、生物学七种视角之间切换，用趣味占卜方式获得可执行建议。

技术栈

HTML + CSS + JavaScript，背景采用 Canvas 静态粒子，无外部 API 依赖，离线也可运行。

设计思路

用“占卜师”叙事降低学习门槛，用“学科方法”提供可落地建议，让知识呈现更有温度、也更有行动价值。

返回占卜

校园文化角落：把课堂知识带回宿舍、社团和日常选择中，才是真正的学以致用。
